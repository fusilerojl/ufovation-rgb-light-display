UFOvation RGB Light Display

> 🎓 Academic Project | Bachelor of Science in Computer Engineering

UFOvation is an Arduino-based RGB LED display project featuring a custom 3D UFO model with synchronized lighting effects. The project was developed to strengthen practical skills in electronics, soldering, circuit design, and Arduino programming. 

Team

This project was completed as part of a team for the Computer Engineering Drafting and Design course.

Features

- RGB LED lighting effects
- Multiple LED animation patterns
- Custom 3D UFO model
- Arduino-controlled lighting
- Soldered electronic circuit

Hardware Used

- Arduino Uno R3
- RGB LEDs
- White LEDs
- Red LEDs
- Resistors
- Dupont wires
